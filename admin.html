📝 設定教學

  1. 開啟 Google Sheets，建立新試算表，複製網址中的 ID
  2. 開啟 Google Drive，建立「商品圖片」資料夾，複製資料夾 ID
  3. 前往 script.google.com，建立新專案
  4. 將 Code.gs 的內容貼入，填入 SHEET_ID 和 FOLDER_ID
  5. 點「部署 → 新部署 → 網頁應用程式」，執行身份選「我」，存取權限選「所有人」
  6. 複製部署網址，貼入上方欄位，按「儲存設定」
  7. 將 customer.html 上傳到任意網頁空間（GitHub Pages / Netlify 皆可）
